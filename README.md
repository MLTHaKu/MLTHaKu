# ESPAÑOL/SPANISH

# Información
Bienvenido al repositorio principal donde se aloja toda la información sobre MLTHaKu.

# Características
* Facilita el proceso e instala el plugin gamesd para el sd2vita en tu PS Vita.
* Sustituye MolecularShell por VitaShell e instala tambien Adrenaline y el instalador de "Henkaku Ensō".
* Utiliza el gamesd modificado que permite montar la memoria oficial como uma0 y la SD2VITA como ux0.
* Puedes realizar la instalación de SD2VITA sin necesidad de usar una memoria oficial.

# Requisitos
* Playstation Vita o Playstation TV con el firmware 3.60 instalado
* SD2VITA con la memoria formateada previamente en exFAT de la manera especial (más información de como formatearla [aquí](https://www.elotrolado.net/wiki/Mods_de_conversores_a_Micro_SD_para_PS_Vita?iframed#Pasos_previos:_Formatear_la_Micro_SD_en_exFAT_.28obligatorio.29)

# Instrucciones

## PS Vita sin haber instalado previamente MLTHaKu.
* Abre VitaShell o MolecularShell y borra la carpeta "tai" de ur0 y ux0
* Quitamos la memoria oficial de la PS Vita con la consola apagada y insertamos el SD2Vita con la Micro SD.
* Encendemos la consola, abrimos el navegador y nos dirigimos al siguiente enlace "http://mlthaku.tk" y seleccionamos SD2VITA.
* Se abrirá un menú, pulsa START para iniciar la instalación.
* Cuando acabe el proceso, abrimos el instalador de "Henkaku ensō" y instalamos "Henkaku Ensō" (si lo tenias previamente instalado, pulsa cuadrado para reinstalar el config y reinicia la consola).
* Cuando se reinicie la vita, ya estará MLTHaKU instalado.

## PS Vita con MLTHaKu previamente instalado
* Apagamos la consola y la encendemos manteniendo L para que no se ejecuten los plugins.
* Abrimos el navegador y nos dirigimos al siguiente enlace "http://mlthaku.tk" y seleccionamos SD2VITA.
* Se abrirá un menú, pulsa TRIANGULO para actualizar MLTHaKu y se reiniciará la consola.
* Una vez reiniciado, ya se habrá actualizado MLTHaKu

# INGLES/ENGLISH

# Information
Welcome to the repository where store the all information about MLTHaKu.

# Features
* Make more easy the process and install the sd2vita plugin in your PS Vita.
* Reemplace MolecularShell for VitaShell and install Adrenaline and the installer of "Henkaku Ensō"
* MLTHaKu uses the modded gamesd that allows mount the oficial memory on uma0 and the SD2VITA on ux0.
* MLTHaKu allows the instalation of SD2VITA wihout the oficial memory.

# Requirements
* Playstation Vita or Playstation TV with the 3.60 Firmware
* SD2VITA with the MicroSD formatted previously on exFAT in the special way (more information [here] (http://wololo.net/2017/07/20/setting-sd2vita-adapter-formatting-microsd-can-done-windows/)

# Instructions

## PS Vita without MLTHaKu installed.
* Open VitaShell or MolecularShell and delete the folder "tai" in "ur0" and "ux0"
* Remove your oficial memory  and insert the SD2VITA with the Micro SD.
* Power on your console, open the browser, insert this link "http://mlthaku.tk" and choose SD2VITA.
* When the menu is opened, press start and the installation process begins.
* When the process finishes, open the installer of "Henkaku ensō" and install "Henkaku ensō" (if you have "Henkaku ensō" previously installed, press square to reinstall the config and reboot your console).

## PS Vita with MLTHaKu previously installed.
* Shutdown your console and power on with holding the L trigger.
* Open the browser, insert this link "http://mlthaku.tk" and choose SD2VITA.
* When the menu is opened, press triangle and the upgrade process process begins.
* When the process finishes, your console are automatically rebooted.
